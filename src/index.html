CSS Cascade Layers Polyfill Demo

What are Cascade Layers?

...a CSS feature that allows us to define explicit contained layers of specificity, so that we have full control over which styles take priority in a project without relying on specificity hacks or !important.

What does this polyfill do?

It uses PostCSS to pre-process the CSS, meaning you can write CSS with Cascade Layers and not worry about it only being available in some of the latest browsers. More details are available in Oddbird's blog post by Sana Javed.
